## 我的理解

什么叫vibe coding

claude code + 语音输入 + 纯自然语言描述，让它全自动化实现特性、修改bugfix。并且很少或几乎不看代码改动，直接全盘接受改动，只要最终代码看着能运行既可，

vibe coding，就是如码，好像写了代码又好像没写

## 技巧

yolo模式打开，让其全自动运行。但是又怕rm -rf之类的操作。
