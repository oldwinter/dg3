## 使用技巧

1. 先创建claude说明文件。本仓库的范例：CLAUDE
2. 复杂问题，使用 plan mode ，切换快捷键 shift + tab
3. 连接到ide，比如cursor。
	1. cursor中安装claude code。
	2. cc中输入/ide进行连接。

## 危险模式，让其全自动驾驶
```
claude --dangerously-skip-permissions
```
